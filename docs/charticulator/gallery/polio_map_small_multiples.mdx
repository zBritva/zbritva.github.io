# A map of Polio incidence rates over time (small multiples)

![A map of Polio incidence rates over time](/images/gallery/polio_map_small_multiples.png)

A chart displaying Polio incidence rates over time for each American state. Each small multiple shows data for a single year.

## Dataset

The number of Polio cases in the United States by Year and State

Retrieved from [Project Tycho](https://www.tycho.pitt.edu/); aggregated into yearly values.
